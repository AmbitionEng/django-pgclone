Documentation
=============

`View the django-pgclone docs here
<https://django-pgclone.readthedocs.io/>`_.

Installation
============

Install django-pgclone with::

    pip3 install django-pgclone

After this, add ``pgclone`` to the ``INSTALLED_APPS``
setting of your Django project.
